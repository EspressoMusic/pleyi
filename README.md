# English Play

פלטפורמה למורה פרטי ותלמיד — מתחברים לחדר ומשחקים משחקי אנגלית בזמן אמת, בעיצוב בסגנון Linky.

## התקנה והרצה

```bash
npm install
npm start
```

פתחו בדפדפן: **http://localhost:3456**

## איך משתמשים

1. **מורה** — לוחץ "אני מורה — צור חדר", מזין שם, מקבל קוד בן 6 ספרות
2. **תלמיד** — לוחץ "אני תלמיד — הצטרף", מזין שם וקוד
3. **מורה** בוחר משחק — שניכם רואים את אותו מסך

## קביעת תור ותשלום

1. **תלמיד** — גולל ל"קביעת תור", בוחר תאריך ושעה, ממלא פרטים ומשלם
2. **מורה** — נכנס ל"יומן מורה" (קוד: `teacher2026`) או בלשונית "יומן תורים" בחדר
3. רואה את כל התורים בזמן אמת — שם, מייל, טלפון, סטטוס תשלום

### הגדרות (קובץ `data/settings.json`)

| שדה | ברירת מחדל |
|-----|------------|
| `teacherPin` | `teacher2026` |
| `pricePerLesson` | 150 ₪ |
| `workingDays` | א'-ה' (0-4) |
| `workingHours` | 09:00–20:00 |

> התשלום כרגע במצב **הדגמה** (מדומה). לחיבור Stripe אמיתי — הגדירו `STRIPE_SECRET_KEY`.

## 5 משחקים

| משחק | תיאור |
|------|--------|
| ⚡ Duel מילים | שאלות אמריקאיות — מי עונה נכון |
| 🧠 זיכרון מילים | התאמת מילה באנגלית לעברית |
| 🎯 איש תלוי | ניחוש אותיות ביחד |
| 🔤 בניית משפט | סידור מילים מבולבלות |
| 🐝 איות | רמז + איות המילה |

## טכנולוגיה

- Node.js + Express + Socket.io — סנכרון בזמן אמת
- HTML/CSS/JS — עיצוב חם בסגנון Linky (Assistant, RTL, מצב כהה/בהיר)
